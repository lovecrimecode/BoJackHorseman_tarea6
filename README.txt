# Proyecto de Gestión de Personajes

Este proyecto es una aplicación web para gestionar personajes, permitiendo realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre una base de datos de personajes.

## Requisitos

Antes de comenzar, asegúrate de tener instalados los siguientes requisitos:

- PHP 7.0 o superior
- Servidor web (Apache, Nginx, etc.)
- MySQL o MariaDB
- Extensión PDO para PHP

## Instrucciones para Configuración y Ejecución

1. **Descomprimir el archivo ZIP**: Extrae el contenido del archivo ZIP en tu servidor web (por ejemplo, en la carpeta `htdocs` de XAMPP o `www` de WAMP).

2. **Ejecutar el asistente de instalación**:
   - Accede a la aplicación a través de tu navegador web: `http://localhost/bojackhorseman_tarea6/install.php`.
   - Completa el formulario con los detalles de tu base de datos. El asistente de instalación se encargará de crear la base de datos y las tablas necesarias automáticamente.

3. **Acceder a la aplicación**:
   - Una vez completada la instalación, accede a la URL principal: `http://localhost/bojackhorseman_tarea6/index.php`.

## Funcionalidades

- Agregar personajes
- Listar personajes
- Actualizar personajes
- Eliminar personajes
- Descargar PDF
